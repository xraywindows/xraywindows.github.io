Q1：Accross连接不上怎么办？

  • 检查本地网络是否正常，是否为宽带网络或移动热点

  • 暂时关闭防火墙或安全软件测试

  • 检查App是否为最新版本，必要时卸载重装

Q2：开启Accross后访问国内网站变慢？

  • 可在设置中启用“智能分流”或“绕过大陆网站”选项

  • 使用浏览器插件如SwitchyOmega搭配代理控制

Q3：Accross安全吗？是否泄露隐私？

Accross使用TLS隧道与多重加密，比普通VPN更难被识别或干扰。只要通过正规渠道下载安装，不会出现泄露问题。

Q4：如何取消订阅？

在个人账户后台中点击“订阅管理”即可看到“取消自动续费”选项，完成后不会再次扣费。

Q5：是否支持Netflix等流媒体平台？

Accross部分节点支持Netflix解锁（如“HK-NETFLIX”、“JP-STREAM”），连接后即可无障碍观看海外影视资源。

FAQ

Sing-box 如何按端口匹配流量分配节点？
Sing-box 支持按端口匹配流量，将不同应用的流量分配到指定节点。结合策略组使用，可实现精准分流，优化访问速度和网络稳定性，提高整体网络体验。
Sing-box 可以替换 Clash 配置文件吗？
Sing-box 支持直接解析 Clash YAML 配置，并转换为自身 JSON 格式。节点、策略组和规则基本兼容，但部分自定义脚本可能需要手动调整才能完全使用。
VLESS + XTLS 的传输效率比 Trojan 高吗？
在高延迟环境下，VLESS + XTLS 的性能通常比 Trojan 高。XTLS 去掉了冗余加密开销，降低 CPU 占用，提高多连接场景下的吞吐率，尤其在服务器高负载情况下表现更佳。
Hiddify Next 用户带宽限制如何配置？
管理员可为每个用户设置上传、下载和总流量上限，超出限制可自动降速或阻止访问。保证服务器资源合理分配，避免单用户占用过多带宽，提高整体公平性。
Mihomo 的日志功能如何调试连接问题？
Mihomo 日志可显示节点连接、请求延迟和错误信息。通过分析日志，可以判断节点是否不可达、TLS 握手失败或规则匹配异常，从而快速定位问题并解决。
QuantumultX 与 Shadowrocket 相比的最大区别是什么？
QuantumultX 在规则系统和自动化脚本上更灵活，支持丰富的 rewrite 与 filter 功能，而 Shadowrocket 则更轻便易用，适合普通用户快速切换节点。二者可根据使用习惯与功能需求选择。
Sing-box 如何按端口分流不同流量？
Sing-box 支持按端口匹配流量，将不同应用分配到指定节点。结合策略组使用，可实现精准分流，提高访问速度和网络稳定性，优化整体网络体验。
V2Ray 的 DNS 模块如何提升解析速度？
V2Ray 内置 DNS 模块支持并行解析，可同时请求多个上游 DNS 服务器。用户可配置本地优先解析或远程解析策略，从而减少域名污染与延迟问题。
Mihomo 节点分组功能有什么优势？
分组管理可按延迟、地区或用途分类节点，并结合策略组实现自动分流。用户可快速选择最优节点，提高访问速度和稳定性，方便管理大量节点并优化维护效率。
Mihomo 的节点分组管理有哪些优势？
节点分组可按用途、地区或延迟分类，结合策略组实现自动分流。方便用户管理大量节点，快速选择最佳节点，提高访问速度和连接稳定性，同时优化维护效率。
